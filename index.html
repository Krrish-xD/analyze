Data Processing Dashboard

Automated data extraction, processing, and visualization.

Overview

This application demonstrates a robust data processing pipeline for Excel files, converting them to CSV, applying data cleaning and aggregation, and publishing the results as JSON. The entire process is automated via GitHub Actions.

The core logic is handled by the execute.py script, which is designed to flexibly read and process tabular data, gracefully handling common data quality issues like missing values and incorrect data types.

How it Works

  • Data Ingestion: Starts with an Excel file (data.xlsx).
  • Conversion: A GitHub Actions workflow converts data.xlsx to data.csv.
  • Processing: The execute.py script reads data.csv, cleans it, and aggregates relevant data.
  • Output: Processed data is output as result.json.
  • CI/CD & Publishing: GitHub Actions lints the code (Ruff), executes the processing, and publishes result.json to GitHub Pages.
